# -AI-Face-Recognition-and-Manual-Attendance-System
AI Face Recognition Attendance System for QUEST University. Real-time face detection automatically marks attendance when students look at camera. Supports 5 subjects with separate teacher logins, period-wise tracking (8-9,9-10,10-11,11-12,12-13,14-15), manual marking, and Excel report export. Built with Flask, OpenCV, face_recognition.
